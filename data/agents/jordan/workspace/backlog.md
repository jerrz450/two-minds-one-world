# Velocity v1 Backlog

## Goal
Deliver an interactive developer productivity dashboard that radically accelerates engineering teams. v1 targets demo-impact, not perfect architecture.

## Core Features (MVP)
1. **Team Dashboard UI**: View active tickets, cycle time, sprint velocity. (Owner: Zoe/Marcus)
2. **Ticket Management**: Create, assign, and update tickets with basic workflow status. (Owner: Priya)
3. **Velocity Metrics**: Display real-time metrics—cycle time, throughput, blockers. (Owner: Marcus)
4. **Alerts System**: Flag stuck tickets, overdue items, and prod incidents. (Owner: Devon)
5. **Simple Auth**: Email login only, session cookie. (Owner: Marcus)
6. **Demo Mode**: Pre-set anonymized data for investor previews. (Owner: Jordan)

## Stretch (Nice-to-Have)
- Filtering by user/tag
- Mobile-friendly layout
- Export metrics as CSV

## Out of Scope (v1)
- SSO, OAuth
- Full audit logging
- Custom integrations (GitHub/Jira)

---
*Goal: Demo-ready in <3 weeks. Prioritize features that show impact in a live walkthrough. Stability is a phase-two concern.*
